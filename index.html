Hi there

I'm NQMVD, a passionate systems programmer and open source developer. I create tools and libraries in Rust, Lua, and other languages with a focus on command-line interfaces, game development, and system utilities.

My work spans across various domains including CLI tools, game development libraries, shell scripting utilities, and system administration tools. I enjoy building efficient, maintainable software that solves real-world problems.

Below are details of some of the projects I have developed, ranging from popular CLI tools to experimental libraries and utilities.

needs

2024

  • Rust
  • CLI Tools
  • System Utils
  • Multi-threading

A command-line tool to check if given binaries are available on your system and optionally get their version information. Built with Rust and featuring multi-threaded processing for efficient system scanning.

This is my most popular project with 16 stars, published on crates.io and actively maintained. It uses parallel processing to quickly scan for installed software across Unix-based systems.
